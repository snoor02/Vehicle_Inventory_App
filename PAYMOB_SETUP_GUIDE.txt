## 🆘 TROUBLESHOOTING

### Issue: Package not found error
- Run: `flutter clean`
- Run: `flutter pub get`
- Run: `flutter pub cache repair`

### Issue: Payment interface not opening
- Make sure credentials are correct
- Make sure Paymob account is active
- Make sure test card is activated (contact support)

### Issue: Transaction declined
- Check if using correct test card: 5123456789012346
- Check expiry date: 12/25
- Check CVV: 123
